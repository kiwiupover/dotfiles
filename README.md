# My Dot files
my dot and atom files


## Symlink Files

```sh
cd ~
ln -shi dotfiles/.atom .atom
ln -shi dotfiles/.gitconfig .gitconfig
ln -shi dotfiles/.zshrc .zshrc
ln -shi dotfiles/.gitignore_global .gitignore_global
ln -shi dotfiles/.githelpers .githelpers
ln -shi dotfiles/.githelpers .githelpers
```
